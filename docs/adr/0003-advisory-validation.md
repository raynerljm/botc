# Setup validation is advisory, never blocking

The bag builder adjusts target counts for setup-modifying characters and warns on deviations, but every warning is overridable and "start game" is always available. Blood on the Clocktower's rule zero is that the storyteller may deviate deliberately, and several characters (Legion, Riot, Atheist, Summoner) break the distribution table by design. Do not "fix" this by making count validation a hard gate.
